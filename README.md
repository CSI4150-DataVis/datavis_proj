## 1. 프로젝트 목적

연세대학교 데이터시각화입문 (CSI4150) 팀 프로젝트

## 2. 설치 방법

다른 라이브러리 설치 없이 localhost로 html 열기

## 3. 사용 방법

1. 지도에서 상권 선택
2. 유동인구, 매출, Sankey 선택 (매출은 서비스 업종 선택해야함)

- 연세대학교, 이화여자대학교는 상권 정보가 존재하지 않아 선택 불가
- 상권이나 서비스 업종을 변경하려면 체크를 전부 해제
